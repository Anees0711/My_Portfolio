LSTM-Based Remaining Useful Life Prediction
EPITA MSc Thesis · Python · TensorFlow · FastAPI · Streamlit
Deep Learning · Time-Series

The challenge was predicting turbofan engine failure from noisy multi-sensor time-series using the NASA C-MAPSS benchmark. I designed LSTM networks with sliding-window generation, dropout, and custom loss functions and benchmarked them against Random Forest, Linear Regression, and SVR baselines. The LSTM achieved R² ≈ 0.87 on held-out data. The full pipeline was deployed as a FastAPI and Streamlit real-time inference prototype.

Embedded IoT Obstacle Detection System
Bahria University · Arduino · ESP32 · Scikit-learn
Embedded ML · IoT Hardware

The goal was to reduce false-positive alerts in a wearable obstacle detection device for visually impaired users. I integrated ultrasonic sensors with Arduino and ESP32 and applied ML-based signal classification to filter noise under real wearable constraints.
